## Формат каждой статьи

- **Длина**: 1500-2500 слов (Google любит «исчерпывающий контент»)
- **Структура**:
  - H1 (1 штука, = заголовок keyword)
  - H2 с разделами (3-6 штук)
  - H3 внутри для подразделов
  - Bullet lists + numbered lists
  - **1-2 скриншота из нашей платформы** (критично для RU-аудитории)
  - Видео вкрапления (YouTube embed — повышает time-on-page)
- **CTA внутри**: «Попробуй стратегию Х бесплатно на CHM Finance →» каждые 500-700 слов
- **Внутренние ссылки**: минимум 3 на другие статьи + 2 на /subscriptions.html
- **Внешние ссылки**: 1-2 на авторитетные источники (docs exchange, Wikipedia для определений)
- **Schema.org**: `Article` + `BreadcrumbList` + `Person` (автор) в JSON-LD

## Расписание публикаций

**Месяц 1**: 4 статьи из High-intent #1-4 (критичные — `kak-nastroit-bota-*`)
**Месяц 2**: 4 Educational #16-19 + 2 Comparison #11-12
**Месяц 3**: 4 Practical #21-24 + 2 из Educational
**Месяц 4+**: добивай оставшиеся; обновляй старые (Google любит «свежий» контент)

**Минимум 1 статья в неделю.** Меньше — Google не верит.

## Метрики для отслеживания

Google Search Console + Яндекс.Вебмастер (бесплатно):
- Impressions в месяц — цель: +50% месяц-к-месяцу
- Avg position — цель: Top 10 для приоритетных keywords за 3 месяца
- CTR — цель: >3% (ниже = проблема с title/description)
- Core Web Vitals — должно быть all green

Google Analytics 4:
- Organic users — базовая метрика роста
- Engagement time > 30 сек — контент интересен
- Bounce rate < 70% — структура удерживает

## Запрещено

- ❌ AI-сгенерированные статьи без редактуры человеком (Google Helpful Content update 2023-2025)
- ❌ Keyword stuffing (засыпание ключевиков) — Google банит
- ❌ Покупка массовых бэклинков (Sape, GoGetLinks) — Google банит
- ❌ Дублирование контента между страницами — каннибализация

## Сигнал что SEO работает

Через 3 месяца в Search Console должно быть:
- **1000+ impressions/день** по твоим keywords
- **30-50 кликов/день** органики
- **3-5 статей в Top 20** по целевым запросам

Через 6 месяцев:
- **10K+ impressions/день**
- **300+ кликов/день** органики
- **5-10 статей в Top 10**

Если через 3 месяца impressions < 300/день → проблема с техбазой или качеством контента. Приходи, разберёмся.
