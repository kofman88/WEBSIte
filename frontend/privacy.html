Конфиденциальность

Политика конфиденциальности

Мы серьёзно относимся к вашим персональным данным и обязуемся защищать вашу конфиденциальность.

Последнее обновление: 7 апреля 2026 GDPR / CCPA совместимо

1 Какие данные мы собираем

  • Telegram ID и username — для идентификации и управления подпиской.
  • API ключи бирж — зашифрованы (AES-256), используются исключительно для авто-трейдинга.
  • Параметры настроек бота — стратегии, тайм-фреймы, риск-параметры.
  • История торговых сигналов — для статистики и улучшения алгоритмов.
  • Технические логи — ошибки, активность, latency.
Что мы НЕ собираем
Email-адреса, реальные имена, платёжные реквизиты карт, паспортные данные, геолокация. Мы не требуем KYC.

2 Как мы используем данные

  • Предоставление и улучшение торговых сигналов и автоматизации.
  • Управление подпиской и доступом к Сервису.
  • Техническая поддержка и диагностика проблем.
  • Анализ агрегированной анонимной статистики для улучшения алгоритмов.
  • Уведомления о важных изменениях в Сервисе.

3 Защита API ключей

Протокол безопасности API
  • Шифрование AES-256 при хранении
  • Передача только по HTTPS/TLS 1.3
  • Ключи никогда не отображаются в логах
  • Доступ только к: Read + Trade | Запрещён: Withdraw
  • Рекомендуем IP Whitelist на бирже

4 Передача данных третьим лицам

Мы не продаём и не передаём ваши персональные данные третьим лицам, кроме:

  • API запросов к биржам (Bybit, Binance, BingX) — только с вашим ключом и от вашего имени.
  • Хостинг-провайдера — исключительно для хранения данных в зашифрованном виде.
  • Требований закона — только по официальному запросу компетентных органов.

5 Хранение и удаление данных

  • Данные хранятся в течение действия подписки + 90 дней после её окончания.
  • Технические логи — 30 дней.
  • По запросу — полное удаление всех данных в течение 72 часов.
  • API ключи удаляются немедленно при отключении автотрейдинга.

6 Ваши права (GDPR / CCPA)

  • Право на доступ — запросить все данные, которые мы храним о вас.
  • Право на исправление — обновить неточные данные.
  • Право на удаление — запросить полное удаление ("право быть забытым").
  • Право на переносимость — получить данные в машиночитаемом формате.
  • Право на возражение — против обработки ваших данных.

Для реализации любого из прав — свяжитесь с нами в Telegram: @crypto_chm

7 Cookie и трекинг

Сайт chmup.top использует минимальный набор технических cookie для корректной работы. Мы не используем рекламные трекеры, пиксели Facebook, Google Analytics или аналогичные системы слежения.
