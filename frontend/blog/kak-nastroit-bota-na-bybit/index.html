Как настроить торгового бота на Bybit за 10 минут

Гайд для новичков8 мин чтенияОбновлено 24 апреля 2026

Если ты торговал на Bybit вручную и хочешь попробовать автоматизацию — эта статья для тебя. Пройдём от нуля до первой сделки бота за 10 минут, по шагам. Без теории про «AI, нейросети и криптомагию» — только то, что реально нужно сделать.

В конце — чек-лист безопасности и разбор 5 самых частых ошибок, из-за которых новички теряют деньги на первом же боте.

Кому подойдёт

Есть аккаунт на Bybit, понимаешь что такое long/short, умеешь ставить стоп-лосс руками. Если это не про тебя — сначала почитай что такое DCA стратегия, потом возвращайся.

Что тебе нужно

  • Аккаунт на Bybit (5 минут, регистрация бесплатна)
  • Аккаунт на CHM Finance (2 минуты — или любая другая платформа с ботами)
  • Немного USDT на Derivatives-кошельке Bybit (рекомендуем $100-500 для старта)
  • Готовность начать с DEMO-режима, а не сразу лить реальные деньги

Шаг 1. Создаём API-ключ на Bybit (3 минуты)

Это самый важный шаг. От того, какие права ты дашь ключу, зависит — спокойно ли ты будешь спать ночью.

  1. На Bybit зайди в профиль → API
  2. Нажми Create New Key → System-generated API Keys
  3. Тип API-ключа: API Transaction
  4. Permissions (это критично):
    • ✅ Contract: Orders, Positions — разрешить
    • ✅ Unified Trading: Trade — разрешить
    • ❌ Withdraw — НЕ разрешать. Никогда.
    • ❌ Transfer — НЕ разрешать
  5. IP Restriction: если твой домашний IP статичный — добавь его. Если нет (мобильный интернет, VPN) — оставь пустым, но следи за логом использования ключа.
  6. Жми Submit, получи пару API Key и Secret

⚠️ Важно

Secret показывается только один раз. Скопируй его в менеджер паролей (1Password, Bitwarden) — если потеряешь, придётся создавать новый ключ.

Шаг 2. Подключаем ключ к CHM Finance (2 минуты)

  1. Войди в dashboard, перейди в Wallet → API Keys
  2. Нажми Add New Key
  3. Биржа: Bybit
  4. Вставь свои API Key и Secret
  5. Label: что-то осмысленное, например «Bybit Unified — USDT фьючерсы»
  6. Нажми Save — система автоматически проверит ключ через тестовый запрос (получит баланс), если ключ валидный — статус Verified ✓

На CHM Finance ключи шифруются AES-256-GCM с индивидуальным IV. Расшифровка происходит только когда бот фактически открывает ордер. В базе данных ключ в открытом виде не хранится ни секунды.

Шаг 3. Создаём первого бота через мастер (3 минуты)

Идём в раздел Bots → Create Bot. Мастер состоит из 6 шагов, они подскажут правильные значения:

3.1 Основы

  • Режим сканирования: «Конкретная пара» (для начала не лезь в market-scanner)
  • Имя бота: например, «BTC SMC 1h»
  • Биржа: Bybit
  • Пара: BTCUSDT (самая ликвидная, минимум проскальзывания)
  • Таймфрейм: 1h — для новичков идеально, сигналов немного, но они качественные

3.2 Стратегия

Для первого бота советую SMC (Smart Money Concepts) или Levels. Они просты для понимания и дают умеренное количество сделок (2-5 в неделю на 1h таймфрейме BTC). Скальпинг — когда захочешь активности и готов платить комиссию. DCA — только если понимаешь философию усреднения.

3.3 Параметры стратегии

Мастер подгрузит настройки по умолчанию — их менять не нужно, они протестированы командой. Если хочешь потюнить — возвращайся после первого месяца торговли со своей статистикой.

3.4 Риск

  • Leverage: 3-5x максимум для новичка. 10x+ — это не трейдинг, это казино.
  • Risk per trade: 1%. Больше — слишком быстро сгоришь при серии лоссов.
  • Max open trades: 2-3. Не раскидывай капитал.

Математика риска

При риске 1% на сделку и серии из 10 убыточных подряд ты потеряешь ~9.5% депозита. При риске 5% на сделку — ~40%. Вот почему опытные трейдеры не превышают 1-2%.

3.5 Бэктест

Критичный шаг — не пропускай. Кликни «Запустить бэктест», система прогонит твои настройки на последних 60 днях свечей Bybit. Смотри на метрики:

  • Win Rate: > 45% — норма для 1R, > 35% — норма для 2R профит-тейка
  • Sharpe Ratio: > 1.0 хорошо, > 1.5 отлично
  • Max Drawdown: не > 20%. Если больше — снижай риск на сделку
  • Profit Factor: > 1.3 — стратегия рабочая

3.6 Режим торговли

DEMO — всегда начинай с него. Минимум неделя. Это бесплатно, бот работает на виртуальных деньгах, но сигналы и исполнение — реальные (по актуальным свечам Bybit).

Шаг 4. Первая неделя — что делать

Теперь бот работает. Твоя задача не лезть в него. Серьёзно. Вот что важно в первую неделю:

  • Ежедневно проверяй: детальный drawer бота → KPI → PnL, количество сделок
  • НЕ отключай бота после первого лосса. 5 подряд лоссов — статистически норма
  • Записывай впечатления: где бот «купил/продал странно», какие сигналы кажутся сомнительными
  • Через неделю — analytics → сравни реальный результат с бэктестом. Если сильно расходится — обычно из-за спреда/проскальзывания на малых таймфреймах

Топ-5 ошибок новичков

  1. Даёт API-ключу право Withdraw. Это катастрофа. Если ключ украдут — уведут все деньги. На CHM Finance мы физически не можем выводить средства, нам это право не нужно.
  2. Плечо 20x+. Одно движение против на 5% → ликвидация. Начинающие трейдеры этот урок учат дорого.
  3. «Улучшает» настройки стратегии без понимания. Например, крутит ATR-множитель стопа. Результат — стратегия перестаёт работать, а причину непонятно.
  4. Запускает 5 ботов одновременно на первой неделе. Потом не может понять, какой из них убыточный. Начинай с 1.
  5. Льёт реальные деньги без DEMO-периода. Экономит неделю — теряет месяц жизни на восстановление психики.

  • ☐ Результаты DEMO в пределах ±15% от бэктеста (если хуже — пересоздай)
  • ☐ API-ключ БЕЗ права Withdraw (проверь второй раз)
  • ☐ На Bybit в балансе только та сумма, которую готов потерять
  • ☐ Ты понимаешь, какой стратегией торгует бот и почему она может слить
  • ☐ Настроены уведомления о сделках (Email или Telegram)
  • ☐ Прочитал риск-дисклеймер
