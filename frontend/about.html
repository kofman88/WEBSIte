Платформа для алгоритмической торговли криптовалютой. Сигналы, авто-трейдинг, аналитика — для розничных трейдеров, которым нужна дисциплина системы вместо эмоций рынка.

Миссия

Устранить главный риск розничной торговли — эмоциональные решения. Даём инструменты, которыми пользуются prop-desks: бэктест, risk-менеджмент, авто-исполнение, мониторинг портфеля. Принцип простой: алгоритм вместо интуиции.

CHM Finance Сигнал-провайдер — публикует алгоритмические торговые сигналы и предоставляет инфраструктуру для авто-исполнения на биржах пользователя. Мы не являемся брокером, не принимаем средства клиентов на торговые счета и не выступаем инвестиционным консультантом. Все торговые операции совершаются напрямую на бирже пользователя через его собственные API-ключи.

Безопасность и соответствие

Активно AES-256-GCM шифрование

API-ключи бирж хранятся в зашифрованном виде с authenticated encryption. Расшифровка возможна только на момент исполнения ордера.

Активно 2FA (TOTP)

Поддержка Google Authenticator / Authy. Рекомендуется для всех аккаунтов.

Активно Read-only + Trade

При создании API-ключа биржи запрашиваем только Read + Trade. Withdraw-разрешение нам не нужно — не включайте его.

Активно GDPR-совместимость

Право доступа, исправления, удаления, переносимости данных. Без трекинг-cookies, без Facebook Pixel / Google Analytics.

Юрисдикционные ограничения

Сервис недоступен резидентам следующих юрисдикций (OFAC / Treasury sanctions и пробелы в лицензировании):

  • • США — требуется брокерская лицензия SEC / CFTC, в разработке
  • • Иран, Северная Корея, Куба, Сирия — санкционные ограничения
  • • Великобритания (retail) — FCA требует pre-authorization для crypto derivatives

Регистрация из этих юрисдикций автоматически отклоняется. Если вы получили блок по ошибке (VPN / неверный geo-IP), напишите в support.
