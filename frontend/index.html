FAQ

Частые вопросы

Безопасно ли подключать биржу?
Да. Мы используем только API ключи с правами на торговлю. Вывод средств через API невозможен. Ключи хранятся в зашифрованном виде (AES-256).
Какие биржи поддерживаются?
Bybit, Binance, BingX и OKX. Мы постоянно добавляем новые биржи.
Какие стратегии доступны?
SMC (Smart Money Concepts) — торговля по институциональным паттернам. Герчик — уровневая стратегия. Скальпинг — быстрые сделки на малых таймфреймах.
Можно ли потерять деньги?
Торговля криптовалютами сопряжена с рисками. Мы используем строгий риск-менеджмент (стоп-лосс, circuit breaker, контроль просадки), но гарантировать прибыль невозможно.
Как работает бэктестинг?
Вы выбираете стратегию, пару, таймфрейм и период. Система тестирует стратегию на исторических данных и показывает метрики: Win Rate, Sharpe Ratio, Max Drawdown и другие.
Есть ли бесплатный тариф?
Да! Бесплатный план включает 3 сигнала в день и 1 активного бота. Этого достаточно чтобы оценить платформу.
